# Career Discovery Self-Assessment Tool

🎯 **Career Discovery** is a professional, AI-ready self-assessment platform designed to help students (particularly 18-year-olds) uncover their true strengths, challenge their career assumptions, and prepare a data-driven profile for AI analysis.

## 🚀 Key Features
- **Structured Journey:** 9 comprehensive sections covering academics, strengths, interests, and research.
- **AI-Driven Logic:** Generates a custom, critical prompt for ChatGPT/Claude/Gemini to provide honest career coaching.
- **Auto-Save:** Progress is saved automatically to `localStorage` so you never lose your data.
- **Progress Export:** Save your progress as a `.json` file and resume later.
- **Professional Export:** Generate a full `.docx` report of your entire assessment.
- **Dynamic UI:** Responsive design with an auto-expanding roadmap and intuitive navigation.

## 🛠️ How to Use
1. **Clone the Repo:**
   ```bash
   git clone https://github.com/[your-username]/CareerGuidance.git
   ```
2. **Launch:**
   Simply open `index.html` in any modern web browser.
3. **Complete the Assessment:**
   Follow the roadmap, providing specific examples for each question.
4. **Get AI Guidance:**
   At the end, click "Copy Full Data & AI Prompt" and paste it into an AI tool (ChatGPT/Claude) for your personalized career analysis.

## 📂 Project Structure
- `index.html`: The main entry point and UI structure.
- `css/style.css`: Modern, responsive styling and animations.
- `js/app.js`: Core application logic, state management, and UI transitions.
- `js/questions.js`: Configuration file containing the full questionnaire structure.
- `js/doc-gen.js`: Logic for generating Word documents using `docx.js`.
